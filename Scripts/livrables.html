Plan d'adressage

Visual Paradigm est un outil de modélisation et de conception qui prend en charge une large gamme de diagrammes utilisés dans la gestion de projets, l'analyse de systèmes, le développement logiciel, et plus encore. Parmi ses nombreuses fonctionnalités, Visual Paradigm permet la création de schémas réseau. Nous avons donc utilisé cet outil pour le dernier point que nous venons de mentionner. Vous pourrez donc voir sur ce schéma le réseau que nous avons crée pour ce projet. Ce schéma contient :

  • Le service de téléphonie : il est assuré par le serveur asterisk. De plus, chaque membre du cabinet dispose d'un téléphone numérique/analogique ou d'un softphone.
  • Aspects informatique, système et réseau :
    • Chaque membre du cabinet paramédical doit disposer d’un ordinateur de bureau câblé, sous Windows 10 avec une identification par login et mot de passe et disposer des applications bureautiques, de navigation Web et de messagerie (Thunderbird)
    • La couverture Wi-Fi du cabinet est assurée par un seul point d’accès Wi-Fi
    • Le plan d’adressage de classe C privée /24
  • Un hyperviseur de type 1 (VMware EXSi 6.7) supporte 2 machines virtuelles :
    • Un contrôleur de domaine sous Windows Server 2019 qui assure la centralisation des comptes utilisateurs et met à la disposition de chaque utilisateur un espace disque dédié. Ce serveur assure également les services de DHCP et de DNS pour le cabinet et pourra supporter certaines applications métiers. Mais aussi, l’application The Dude. Cette application gratuite permet une vue globale sur tous les éléments d’un réseau afin d’en assurer la supervision en temps réel.
    • Un serveur GNU/Linux sous Ubuntu est nécessaire pour héberger certaines applications métiers et plusieurs services (dont : Web, MySQL, FTP, TFTP, asterisk, SMTP, POP3).
