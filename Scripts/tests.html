Test de connectivité interne (kinésithérapeute vers secrétaire)

    • Adresse IP : 192.168.2.71/24
    • Passerelle par défaut : 192.168.2.254
  • Information PC secrétaire :
    • Adresse IPV4 : 192.168.2.53/24

  • RXE104-Kiné2
  • RXE104-Kiné3
  • RXE104-Généraliste
  • RXE104-Dentiste
